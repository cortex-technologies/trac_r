# trac_r

A cloud-native, lazy-loading ML experiment tracker backed by Hugging Face Datasets.

## Installation
```bash
pip install git+https://github.com/yourusername/trac_r.git
```

or if you have uv installed we recommend

```
uv add git+https://github.com/your-username/trac_r.git
```

## Usage
Launch the dashboard via:
```bash
trac-r
```
Or stream from the cloud:
```bash
trac-r --repo-id yourname/dataset
```
